- Number Functions (ROUND, ABS)

- NULL Functions (COALESCE, ISNULL, NULLIF)

- Views, CTAS, Temp Tables
- Indexes (Clustered, Nonclustered, Columnstore)
- Stored Procedures & Triggers
- Data Warehouse Project (Bronze / Silver / Gold layers)
- Exploratory Data Analysis (EDA) Project
